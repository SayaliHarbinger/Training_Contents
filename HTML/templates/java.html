Java Introduction

Java is a high-level programming language developed by Sun Microsystems and currently owned by Oracle. It was designed to be platform-independent and allows developers to write code that runs on any operating system or platform that has a Java Virtual Machine (JVM). Java is widely used for building a variety of applications, including desktop, web, mobile, and enterprise applications.

Java Syntax

Java syntax refers to the set of rules that define how Java code should be written. It includes guidelines for declaring variables, defining classes and methods, using control flow statements, and more. Here's an example of a simple Java program:

public class HelloWorld {
  public static void main(String[] args) {
    System.out.println("Hello, World!");
  }
}

Java Output

When you run a Java program, the output is typically displayed in the console or terminal window. For example, if you run the "HelloWorld" program mentioned earlier, the output will be:

Hello, World!

Java Comment

In Java, comments are used to add explanatory notes or disable certain sections of code. There are three types of comments in Java:

  • Single-line comments: These start with "//" and continue until the end of the line.
  • Multi-line comments: These start with "/*" and end with "*/". They can span multiple lines.
  • Documentation comments: These start with "/**" and end with "*/". They are used to generate documentation using tools like Javadoc.

Here's an example:

// This is a single-line comment

/*
This is a multi-line comment.
It can span multiple lines.
*/

/**
 * This is a documentation comment.
 * It provides information for generating documentation.
 */
